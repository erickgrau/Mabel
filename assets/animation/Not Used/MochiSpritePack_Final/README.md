# Mochi Sprite Pack

Drop this folder into the local Mabel repo.

## Files

- `Assets/Mochi/Sprites/mochi_walk_16_clean.png`
  - 8192 x 512
  - 16 frames
  - 512 x 512 per frame
  - PNG RGBA with true alpha

- `Assets/Mochi/Sprites/frames/`
  - individual transparent PNG frames

- `Assets/Mochi/Sprites/mochi_walk_preview.gif`
  - quick preview

- `Assets/Mochi/mochi_animation_config.json`
  - animation metadata

- `Sources/Mochi/MochiNode.swift`
  - SpriteKit behavior node

- `Sources/Mochi/MochiScene.swift`
  - minimal scene wrapper

## Mochi behavior

- walks s l o w
- slight hesitation every few steps
- gentle, anxious personality
- subtle breathing
- rare ear twitch
- cursor-aware look
- rain-mode shiver
