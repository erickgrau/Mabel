Engine

Transcription engine
Local runs entirely on this Mac. Cloud uses your own Groq key.
Whisper model
Larger models are more accurate, take more memory and time.
Groq API key
Stored in macOS Keychain. Get a key at console.groq.com.

Privacy

✓
Audio is deleted after transcription
Recordings are written to a temp file, transcribed, and removed.
✓
Transcripts are never logged
Mabel never writes the text of your dictation to disk or sends it anywhere.
✓
No telemetry, no analytics
No usage tracking, no phoning home.
✓
Keys live in the OS Keychain
Your Groq key, if you provide one, is never written to a config file.
✓
Local stats are counts only
Dictation counts and dates, never content.
